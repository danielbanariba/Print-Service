INSTALACIÓN:
─────────────────────────────────

    1) Hacer DOBLE CLIC en: INSTALAR_SERVICIO.bat
    2) Aceptar permisos de administrador cuando Windows pregunte

¡LISTO! El servicio ya está instalado y funcionando

════════════════════════════════════════════════════════════════════

VERIFICAR INSTALACIÓN:
─────────────────────────
  • Hacer doble clic en: VERIFICAR_ESTADO.bat
  • Debe mostrar:
    - ✓ Servicio instalado

════════════════════════════════════════════════════════════════════

DESINSTALACIÓN:
──────────────────
  • Hacer doble clic en: DESINSTALAR_SERVICIO.bat
  • Presionar [S] para confirmar

════════════════════════════════════════════════════════════════════

UBICACIÓN DEL SERVICIO:
──────────────────────────
  El servicio se instala en: C:\Print-Service\TSSPrint.exe
  
  Nombre del servicio: TSSPrintService

════════════════════════════════════════════════════════════════════

NOTAS IMPORTANTES:
─────────────────────
  • El servicio se inicia automáticamente con Windows
  • Compatible con impresoras térmicas y de etiquetas
  • No requiere configuración adicional
  • Los logs se pueden ver en el Visor de Eventos de Windows
